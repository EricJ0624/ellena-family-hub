# Google Maps API 사용 정책

## 플랫폼별 SDK

- **웹**: Maps **JavaScript API** 사용  
  - 스크립트: `https://maps.googleapis.com/maps/api/js?key=...&libraries=places,marker`
- **앱(모바일)**: Maps **SDK for Mobile** 사용  
  - Android: Maps SDK for Android  
  - iOS: Maps SDK for iOS  

나중에 앱용 코딩으로 변환할 때 웹(JavaScript API) 대신 위 Mobile SDK로 전환할 것.
